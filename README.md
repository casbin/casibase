## Architecture

Casibase contains 2 parts:

| **Name**       | **Description**                                   | **Language**                            |
|----------------|---------------------------------------------------|-----------------------------------------|
| Frontend       | User interface for Casibase                       | JavaScript + React                      |
| Backend        | Server-side logic and API for Casibase            | Golang + Beego + Python + Flask + MySQL |
